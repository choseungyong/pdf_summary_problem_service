## 요약 정리본

### 짧은 개요
본 강의는 컴퓨터 네트워크의 기본 개념과 구조를 이해하는 데 중점을 두며, 다양한 네트워크 계층과 데이터 전송 방식을 학습합니다.

### 핵심 개념 목록
- 데이터 통신
- 네트워크 유형 (LAN, WAN)
- 프로토콜 계층화
- TCP/IP 모델
- OSI 모델
- 데이터 흐름 방향 (단방향, 반이중, 전이중)

### 중요 공식/정의
- 데이터 통신: 두 장치 간 데이터 교환
- 프로토콜: 통신 규칙의 집합
- 네트워크의 성능 기준: 전달 시간, 응답 시간, 처리율

### 자주 하는 실수/오해
- LAN과 WAN의 차이를 혼동하는 경우
- OSI 모델과 TCP/IP 모델의 계층 구조를 혼동하는 경우
- 데이터 흐름 방향의 개념을 잘못 이해하는 경우
